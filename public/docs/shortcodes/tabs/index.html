    Tabs #

    Tabs let you organize content by context, for example installation instructions for each supported platform.

{{% tab "Linux" %}} # Linux Content {{% /tab %}}
{{% tab "Windows" %}} # Windows Content {{% /tab %}}
{{% /tabs %}}
